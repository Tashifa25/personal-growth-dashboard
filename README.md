# Personal Growth Tracking Dashboard
A simple front-end web application built using HTML, CSS, and Vanilla JavaScript.

## Features
- Add evaluation sessions
- 5-star rating system
- Automatic overall score calculation
- Motivational feedback based on score
- LocalStorage persistence
- Delete session functionality
- Clean responsive UI

## Tech Stack
- HTML
- CSS
- JavaScript (Vanilla)
- LocalStorage API

## Purpose
This project was built to practice:
- DOM manipulation
- Event handling
- LocalStorage usage
- UI structuring
- Basic state management

## Future Improvements
- Data visualization charts
- Export to PDF
